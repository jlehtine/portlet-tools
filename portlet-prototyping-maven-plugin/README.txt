JETTY-PLUTO MAVEN PLUGIN
========================

A Maven plugin for rapid portlet development using the Jetty servlet
container and the Apache Pluto portlet container.

For more information, see:

  http://www.jlehtinen.net/portlet-tools/jetty-pluto-maven-plugin/

For more information on Jetty and Apache Pluto, see:

  http://jetty.codehaus.org/jetty/
  http://portals.apache.org/pluto/
